# Google Workspace Personas

Umbrella skill for role-based operating patterns built on `gws`.

## Shared persona workflow
1. Clarify the role’s primary inbox/calendar/docs surfaces.
2. Start from the role’s default morning scan: agenda, inbox, pending tasks, and relevant files.
3. Use one of the workflow helpers when possible.
4. Make communications concise and role-appropriate.
5. Confirm before sending or changing attendee-visible state.

## Persona families
### Executive / coordination
- Executive assistant
- Project manager
- Event coordinator
- Team lead

### Communication / support
- Customer support
- Sales ops
- Content creator
- Researcher

### Administrative / operations
- HR coordinator
- IT admin

## Default tool combinations
- Inbox-heavy roles: Gmail + Chat + Calendar
- Documentation-heavy roles: Drive + Docs + Sheets
- Coordination-heavy roles: Calendar + Meet + Tasks + workflow helpers

## Reusable role habits
- Start with calendar and inbox triage.
- Use workflow helpers for meeting prep, standups, weekly digests, and email-to-task conversion.
- Keep an audit trail in Docs/Sheets for follow-up-heavy work.
