# Persona patterns

## Executive / coordination
- exec assistant
- project manager
- event coordinator
- team lead

Shared pattern: agenda scan -> inbox triage -> meeting prep -> docs and follow-up.

## Communication / support
- customer support
- sales ops
- content creator
- researcher

Shared pattern: inbox or request queue -> working doc/sheet -> outbound updates.

## Admin / operations
- HR coordinator
- IT admin

Shared pattern: policy or incident context -> system of record -> user-facing comms.
