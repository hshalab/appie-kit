# CKM Visual System

## Overview
Use this umbrella for CKM-flavored visual work: brand direction, design systems, presentation decks, banners, and UI styling. Prefer a single coherent visual language over one-off asset-specific rules.

## Core Modes
- **Brand:** voice, identity, palettes, logo-adjacent cues, and usage rules.
- **Design system:** tokens, spacing, typography, components, and reusable patterns.
- **Slides:** narrative decks, strategic layouts, charts, and presentation polish.
- **Banner design:** social headers, campaign graphics, and hero imagery.
- **UI styling:** product interfaces, page sections, and component-level aesthetics.

## Workflow
1. Determine the asset type and audience.
2. Pick one visual direction and keep it consistent.
3. Reuse shared tokens, typography, and composition rules across outputs.
4. Check contrast, hierarchy, and safe zones before finalizing.

## Pitfalls
- Don’t split brand, slide, and UI work into separate mental models when they share the same system.
- Don’t default to generic layouts or typography.
- Don’t let banners, decks, and product UI drift stylistically.
