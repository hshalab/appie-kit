## Thumbnail Design Rules (Based on Research + Channel Audit)

### The Formula
```
Expressive Face (40% of frame) + High Contrast Colors + Bold Text (3-5 words) + Brand Element = High CTR
```

### Specs
- **Dimensions:** 1280 x 720 px (16:9)
- **Format:** JPG (quality 85-90%) or PNG
- **Max file size:** 2 MB
- **Color profile:** sRGB

### Weblyfe Brand System
- **Primary font:** Rethink Sans (800 weight for headlines, 700 for subtext)
- **Colors:**
  - Background dark: #031D16 (Green Goblin)
  - Background mid: #0E3D31 (Storm Green)
  - Accent: #247459 (Emerald)
  - Gold: #DFB771 (primary accent/highlight)
  - Gold light: #FFD99A (Turmeric)
  - Text: #F6FEFC (Snow white)
  - Alert/emphasis: #FF3B3B (red for urgency)
- **Brand mark:** "WEBLYFE" text in gold, bottom-right corner, Rethink Sans 600
- **No em dashes in any text. Ever.**

### Composition Rules
1. **3-Zone Layout:** Left 40% = Face/Subject | Center 30% = empty/gradient | Right 30% = Text
2. **Alternative:** Split layout. Left = text stack. Right = face. Works for client stories
3. **Face size:** Minimum 40% of frame height
4. **Text:** 3-5 words max. 100-200px font size at full res
5. **Safe zones:** Nothing in bottom-right corner (YouTube timestamp overlay)
6. **Mobile test:** Must be readable at 168x94 pixels

### Text Rules
- **Max 5 words** on any thumbnail
- **Font weight:** 800 (Extra Bold) minimum
- **Outline/shadow:** 3px dark stroke or drop shadow for readability
- **Gold highlight** on the key number or word
- **No full sentences.** Fragments only. "5 FIG to 7 FIG" not "How we went from five figures to seven figures"

### Face Rules
- **Use the close-up white t-shirt photo** (pic1.jpg) as default base
- **Techwiz sey photo** for tech/AI content
- **Expression bank needed:** Shoot 10-15 expressions (surprised, focused, excited, pointing, thinking)
- **Background removal:** Always. Face floats on branded background
- **Glow/outline:** Subtle emerald or gold glow around cutout (2-3px)

## Thumbnail Templates (5 Types)

### Template 1: Client Success Story
**Use for:** Student/client result videos (most of channel)
```
Layout: Split 50/50
Left: Client's face (cutout, with glow)
Right: Revenue number in HUGE gold text
       "in 30 DAYS" or similar timeframe below in white
Bottom-right: Weblyfe gold watermark
Background: #031D16 to #0E3D31 gradient (left to right)
Accent: Gold diagonal line or arrow from left to right
```
**Example text:** "$0 to $8K" / "30 DAYS"

### Template 2: Educational/How-To
**Use for:** Tutorials, AI workflows, tech content
```
Layout: 60/40
Left: Seyed's face (cutout, expressive, pointing or gesturing)
Right: Short punchy text in white, key word highlighted gold
Background: #0E3D31 solid or subtle gradient
Element: Relevant icon or screenshot (small, 20% of frame)
Bottom-right: Weblyfe gold watermark
```
**Example text:** "AI SALES AGENT" / "20 MIN BUILD"

### Template 3: Interview/Podcast
**Use for:** Roslan, guest conversations, AUD Dubai
```
Layout: Dual face
Left 35%: Guest face (cutout)
Right 35%: Seyed's face (cutout)
Center 30%: Text stack
Background: Dark gradient with subtle emerald accent
Both faces: Matching gold outline glow
Text: Guest result or topic in white, number in gold
Bottom-right: Weblyfe gold watermark
```
**Example text:** "$50K to $1M+" / "ROSLAN"

### Template 4: Personal Brand/Vlog
**Use for:** Lifestyle, travel, The Weblyfe series
```
Layout: Full-bleed photo with overlay
Base: Cinematic photo (real location, not studio)
Overlay: Dark gradient from bottom (60% opacity)
Text: Bottom-left, large white text, 3-4 words
Accent: Gold underline or highlight bar
Bottom-right: Weblyfe gold watermark
```
**Example text:** "MOST DANGEROUS NIGHT" / "CAPETOWN"

### Template 5: Series (Creative OS, Weblyfe University)
**Use for:** Multi-part series content
```
Layout: Centered, symmetrical
Background: Signature series background (cosmic for Creative OS, emerald for WU)
Text: Episode number (gold) + Topic (white)
Element: Series logo/icon
Bottom-right: Weblyfe gold watermark
Consistent: Every episode uses exact same layout, only text changes
```

## Thumbnail Creation Flow

### Step 1: Prepare Assets
1. Select face photo from `/tmp/seyed-photos/` bank
2. Remove background (rembg or manual)
3. Apply subtle gold glow outline (3px)
4. Save as PNG with transparency

### Step 2: Generate Base
```bash
# Using the HTML template generator
node projects/weblyfe-ai/content/thumbnails/generate.js \
  --template "client-success" \
  --face "./faces/seyed-closeup.png" \
  --text-main "$0 to $8K" \
  --text-sub "30 DAYS" \
  --output "./output/thumbnail-samouil.png"
```

### Step 3: Review Checklist
- [ ] Readable at 168x94 pixels (mobile size)?
- [ ] Face takes up 40%+ of frame?
- [ ] Text is 3-5 words max?
- [ ] Rethink Sans font used?
- [ ] Gold accent on key word/number?
- [ ] Weblyfe watermark bottom-right?
- [ ] No text in bottom-right (timestamp zone)?
- [ ] High contrast against both light and dark YouTube modes?
- [ ] No em dashes anywhere?
- [ ] Does it match the video content? (no clickbait mismatch)

### Step 4: Export
- Save as JPG, quality 85-90%
- Verify under 2 MB
- Upload to YouTube via Studio or API

### Step 5: A/B Test (Optional)
- YouTube Test & Compare: upload 2-3 variants
- Let run 7-14 days
- Check CTR in YouTube Analytics
- Winner becomes the template for similar content

## Writing Rules (All Copy)
- **Never use em dashes** (no long dashes)
- Use commas, periods, or line breaks instead
- "It is" instead of "it's" in thumbnails (readability)
- Numbers always as digits: "7 FIGURES" not "seven figures"
- Currency with symbol: "$10K" or "9K" (not "9,000 euros")
