## Patterns Van Sites of the Day

### 1. MOTION AS MEANING
Award winners animate met PURPOSE:
- Hover states die informatie onthullen
- Scroll-triggered reveals die verhaal vertellen
- Transitions die flow creëren

**Niet:** Random animations die afleiden
**Wel:** Motion die content ondersteunt

### 2. TYPOGRAPHY AS HERO
Vaak is tekst het visuele element:
- Massive headlines (100px+)
- Custom fonts / variable fonts
- Typography as imagery
- Text animations (letter by letter reveals)

### 3. UNCONVENTIONAL LAYOUTS
Breaking the grid met intentie:
- Overlapping elements
- Asymmetric compositions
- Full-bleed sections
- Horizontal scrolling (strategic)

### 4. IMMERSIVE EXPERIENCES
De beste sites voelen als:
- Je stapt een wereld binnen
- Er is een "wow" moment
- Je wilt scrollen om meer te zien
- Je herinnert je de site

### 5. PERFORMANCE + BEAUTY
Winners zijn OOK:
- Fast loading
- Mobile responsive
- Accessible
- Technically solid

1. **Design** — Is het visueel excellent?

## Patterns To Study & Implement

### Quick Wins
| Pattern | Hoe implementeren |
|---------|-------------------|
| Smooth scroll | Locomotive Scroll / Lenis |
| Reveal animations | GSAP ScrollTrigger |
| Custom cursor | CSS + JS, on-brand |
