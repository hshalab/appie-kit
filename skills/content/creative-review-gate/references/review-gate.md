# Review Gate for Appie-style Reel Production

This reference captures the concrete review-pack pattern that emerged during the Appie origin-story reel workflow.

## What to deliver before continuing
- Script draft with scene-by-scene beats
- Starting frames / keyframes for the first pass
- A short note on which frame is the character anchor
- A short note on which assets are locked vs editable
- CTA / ending line
- Optional SFX / narration timing notes

## What to pause on
Stop before:
- motion generation
- clip stitching
- final export
- caption finalization if the user is still curating story beats

## Useful framing
When the user asks for oversight, lead with a compact review pack rather than a full production dump:
- 1 sentence on the concept
- 3 to 7 bullets for the story beats
- the keyframes/contact sheet
- a direct question asking what to keep, change, or approve

## Character continuity checklist
- Keep the opener anchored to the user's actual likeness when requested
- Preserve the canonical character palette and silhouette across scenes
- Mark any frame that is an approved anchor so later steps can reuse it
- If one outfit or pose is approved, keep adjacent scenes aligned to that choice unless the user asks otherwise

## Editing principle
If the user asks for one specific revision, change the smallest adjacent set of assets and re-present the review pack instead of restarting the whole sequence.
