# Content Calendar — Weekly & Monthly Planning Framework

## 2026 Best Practices

- **Batch create** content in dedicated sessions (reduces burnout)
- **Plan format mix** before filling in dates
- **Rotate hashtag groups** per post
- **Track engagement** per format to optimize over time
- **Plan Stories separately** (24-hour shelf life)
- **Grid preview** before scheduling (visual consistency)

## Content Batch Creation Workflow

### Step 1: Weekly Planning (30 min)
1. Review last week's engagement data (Zernio)
2. Set this week's theme/pillars
3. Fill in format mix (carousel/Reels/single)
4. Assign posting times

### Step 2: Content Batch Session (2-3 hours)
1. Create all hooks first (copy from `references/viral-hooks.md`)
2. Write all captions (NL + EN for each post)
3. Generate/create all visuals
4. Save to `$CONTENT_ASSET_DIR/captions/[week-X]/`

### Step 3: Scheduling (30 min)
1. Load into Meta Business Suite or scheduling tool
2. Set exact times per NL+EN schedule
3. Update calendar status
4. Verify grid preview (visual consistency)

### Step 4: Engagement (Daily)
1. Respond to all comments within 1 hour
2. Engage with similar accounts
3. Monitor early engagement velocity
4. Edit caption if needed (first 30 min)

### Step 5: Weekly Review (30 min)
1. Review metrics per format
2. Identify top-performing post
3. Adjust next week based on data
