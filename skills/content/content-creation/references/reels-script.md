# Reels Script Framework — 2026 Best Practices

## Why Reels Win in 2026
- **36% more reach** than carousels (discovery focus)
- Widest algorithmic distribution
- Hook-first scripting critical (71% decide in 3 sec)
- 70%+ completion = algorithmic boost

## Reels Structure (0-30 seconds)

### 5-Part Framework
```
Hook (0-3s)     → Pattern interrupt, stop scroll
Promise (3-5s)  → "Here's what I have to show you"
Proof (5-8s)    → (optional) Quick credibility
Steps (8-25s)   → Core value delivery
CTA (25-30s)    → Specific next action
```

### 3-Scene Framework (Alternative)
```
Scene 1 — Hook (0-3s)
Scene 2 — Body (3-22s)
Scene 3 — CTA (final 5s)
```

## Hook Templates (Copy-Paste)

### Pattern Interrupt Hooks
- "Stop if you want [specific result]"
- "The biggest mistake [audience] makes when [action]"
- "Here's what we learned after [specific experience]..."
- "Did you know that [surprising fact]..."
- "Nobody talks about [topic]. Let me change that."

### Outcome Hooks
- "In 30 seconds, you'll know [outcome]..."
- "This [X]-minute [task] just changed my [area]..."
- "I'm about to show you something most people never figure out..."
- "By the end of this reel, you'll know [specific thing]."

### Curiosity Hooks
- "The interview question that predicted our best hires..."
- "I never told anyone this about [topic]..."
- "The one thing nobody tells you about [topic]..."
- "Wait for the end — this is the part that matters most."

## Reels Caption Formula
```
[Hook line — 1-2 sentences, stop scroll]

[Body — expand on the value, tell the story]

[CTA] Follow for more / Save this / Comment your [X]

[Hashtags — 3-5, EN or NL]
```

## Platform-Specific Rules (2026)

### Instagram Reels
- 3-second hook minimum
- Polished aesthetics + authentic content
- Under 30 sec for new audiences
- 30-90 sec for existing followers
- Text overlay from frame 1 (boosts completion)

### TikTok Reels
- 2-second hook (faster pacing)
- 20% higher energy
- Trending sounds
- Cross-post WITHOUT watermark (40-60% reach penalty otherwise)

### YouTube Shorts
- 3-5 second hooks tolerated
- Searchable titles matter
- Longer retention = recommended

## Reels Best Practices
- Always include captions (start of video, not end)
- Vertical 9:16 format (1080x1920)
- First frame must make sense without sound
- Face presence increases engagement
- Fast cuts > long static shots
- Hook must deliver on promise (no bait-and-switch)

## Engagement Triggers for Reels
- Questions in caption (drives comments)
- "Tag someone who..." (shares)
- "Save this for later" (saves)
- "Follow for more..." (followers)
- Controversial opinions (comments)

## Reels vs Carousels Decision Matrix
| Goal | Format |
|------|--------|
| Discovery, new followers | Reels |
| Saves, reference content | Carousel |
| Deep education | Carousel |
| Quick entertainment | Reels |
| Building authority | Both |
| PDF/offer conversion | Carousel |
