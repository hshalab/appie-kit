# Viral Hooks Reference — 75+ Copy-Paste Hooks (2026 Data-Backed)

## Hook Psychology (Why They Work)

The first 3 seconds decide everything. Instagram's algorithm measures **engagement velocity** in the first 30-60 minutes. A strong hook:
1. Triggers curiosity or emotion (stop the scroll)
2. Creates a micro-commitment (you've already invested 1 second)
3. Promises value for the time cost

## Hook Format Guide

|--------|-----------|---------|
| **Carousel** | 1-3, 31-37, 66-72 | Educational, lists, transformation |
| **Reels** | 4-7, 14-20, 51-57 | Time contrast, story, POV |
| **Single Image** | 21-30, 58-65 | Bold statements, challenges |
| **Stories** | 41-50, 62-65 | Relatable, interactive |

## 2026 Algorithm Priority Hook Elements
- **DM shares** (strongest signal in 2026) → hooks that make people think "my friend needs to see this"
- **Saves** → hooks that promise lasting value (educational, reference content)
- **Comments** → hooks that create opinion or invite disagreement
- **Watch time** → hooks that create tension requiring resolution

## NL vs EN Adaptation Rules
- NL: Direct, "je" instead of "your", "jij" instead of "you"
- EN: More expansive, "the", "your", full explanations
- Never mix language in hashtags
- Hook same in both, caption adapts
