# SEO Skill Pack - Index

Source: Seyed's Weblyfe University Tornado Masterclass (6 modules, downloaded 2026-05-03).

## Skills

| Skill | Path | Summary | When to use |
|-------|------|---------|-------------|
| Bezoekersmagneet | `skills/seo-bezoekersmagneet/SKILL.md` | SEO fundamentals: why it matters, Google's 6 ranking factors, CTR data by position, reputation management. | Client onboarding, justifying SEO investment, explaining organic vs paid. |
| Keyword Strategie | `skills/seo-keyword-strategie/SKILL.md` | Long-tail keyword selection with Ubersuggest/SEMrush, 70/30 search volume split, monsterartikel format, keyword placement rules. | Before writing any content, auditing a page that is not ranking. |

## Suggested workflow for a new Weblyfe client site

1. Bezoekersmagneet: set expectations with client, confirm GA4 + Search Console are connected.
2. Technische SEO: fix crawl issues and submit sitemap before any content work.
3. Keyword Strategie: map keywords to URLs before writing a single word.
4. Smart Content: plan content clusters and internal link architecture.
5. SEO Checklist: run on every page before publish.
6. Backlinks: start outreach after at least 5 pieces of strong content are live.
